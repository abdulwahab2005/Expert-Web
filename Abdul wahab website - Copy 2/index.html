Html

HTML is the standard markup language for creating Web pages. It describes the structure of a Web page and consists of a series of elements. HTML elements tell the browser how to display the content.

CSS

CSS is used to describe the presentation of a document written in HTML or XML. You can use CSS to control the layout of your Web page, including colors, fonts, and more.

javaScript

JavaScript (JS) is a programming language used to create interactive effects within Web browsers. You can use JS to create dynamic effects such as image rollovers and interactive forms.

jquery

jQuery is a JavaScript library that makes web development easier and faster. It helps you to traverse and manipulate HTML documents, handle events, animate elements, and perform Ajax requests with a simple and consistent API that works across different browsers
